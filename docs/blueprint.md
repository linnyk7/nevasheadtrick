# **App Name**: NEVASFZ HEADTRICK

## Core Features:

- User Authentication: Secure login interface enforcing user validation rules (username ending with 'fz', password '0') and providing immediate feedback.
- Function Toggle Control: Interactive, toggle-style buttons for activating and deactivating core functionalities like '0 Delay' and 'Ajuste de Mira'.
- Dynamic Function Feedback: Displays loading animations and success messages to confirm the activation status of each panel function.
- Responsive Layout Adaptation: Ensures the entire interface seamlessly adjusts and functions optimally across various screen sizes, including mobile devices.
- Smooth UI Transitions: Implements fluid visual transitions between the login page and the main dashboard, enhancing user experience.

## Style Guidelines:

- Primary color: Electric Purple (#B06CFF) to establish the futuristic, neon aesthetic. This vibrant hue creates focus and excitement on the dark background.
- Background color: Deep Shadow Black (#271F2E), a dark base with a subtle purple undertone to enhance depth and provide a sleek, immersive feel, complementing the neon elements. This background provides a subtle visual cue that there's 'something' there and provides the anchor for glass-morphism.
- Accent color: Cybernetic Blue (#424EBA), a rich, deeper blue that provides an analogous contrast to the primary purple. This can be used for secondary interactive elements or highlighted text, working well with implied metallic textures.
- Metallic accents: Implement gradients and textural overlays simulating polished silver on buttons and key elements, integrated with the existing color palette for a sophisticated 'futuristic/gamer' feel.
- Font choice: 'Space Grotesk' (sans-serif) for both headlines and body text. Its technical and modern geometric feel aligns perfectly with a futuristic and gamer-oriented aesthetic.
- Icon style: Utilize minimalist, vector-based icons rendered in neon purple. Inspirations include simplified crosshairs, stylized lightning bolts, abstract control panel symbols, and other gaming motifs.
- Layout design: A clean, spacious dark layout with strategically placed 'glass-morphism' elements for transparency and depth, providing an immersive, high-tech panel experience. Emphasize responsive design for fluid adaptability across all devices.
- Animation: Integrate smooth, subtle animations for UI elements. This includes soft transitions between screens, dynamic hover effects on buttons with a neon glow, and fluid loading animations (spinners/bars) in electric purple.
